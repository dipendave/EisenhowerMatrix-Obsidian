# Eisenhower Matrix for Obsidian

A plugin that adds an interactive Eisenhower Matrix to Obsidian for prioritizing tasks by urgency and importance.

![Obsidian](https://img.shields.io/badge/Obsidian-0.15.0+-purple)

## Features

- **4-quadrant matrix** — Do, Schedule, Delegate, Eliminate
- **Drag-and-drop** — move tasks between quadrants (desktop + mobile)
- **Due dates** — relative formatting (Today, Tomorrow, etc.) with overdue highlighting
- **Mobile support** — long-press drag, responsive layout
- **Theme-aware** — works with light and dark mode

## Installation

1. Open **Settings** > **Community plugins** in Obsidian
2. Turn off **Restricted mode** if prompted
3. Click **Browse** and search for **Eisenhower Matrix**
4. Click **Install**, then **Enable**

## Usage

Open the matrix from the ribbon icon (grid) or via command palette: **Open Eisenhower Matrix**.

Add tasks with the **+** button in each quadrant. Drag tasks between quadrants to re-prioritize.
